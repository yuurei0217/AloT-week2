# Cheng Jen Chen - Personal Portfolio Page

A beautiful, single-page personal portfolio and CV website designed with a modern split-screen aesthetic. This project was developed as a clean, simple landing page.

## Features

* **Modern Design:** Clean, premium aesthetic with smooth hover animations, sleek typography, and shadows.
* **Split Layout:** A two-pane side-by-side design inspired by top-tier modern CV templates.
* **Live Clock:** A dynamic, real-time updating digital clock showing the current time.
* **Responsive Layout:** The page design seamlessly adjusts and stacks for smaller devices and mobile screens.

## Technologies Used

* **HTML5:** Semantic document structure.
* **CSS3:** Custom CSS variables, Flexbox layouts, transition effects, and keyframe animations. It uses the `Outfit` font family from Google Fonts.
* **JavaScript:** Vanilla JavaScript handles the precise timing and formatting of the localized real-time clock.

## Project Structure

* `index.html` - The HTML skeleton and content of the page.
* `index.css` - The styles dictating the visual appearance and responsive design.
* `index.js` - The frontend logic for updating dynamic elements over time.

## How to View

You don't need any complex development environment to view this site:
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/yuurei0217/AloT-week2.git
   ```
2. Open the directory and double click on `index.html` to run it in your default web browser.
